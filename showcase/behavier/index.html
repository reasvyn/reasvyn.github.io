Why behavier

Built for developers
who care about precision

Drop-in user behavior tracking without the overhead of bloated analytics SDKs. Precise, composable, and entirely under your control.

Zero dependencies

Pure browser APIs, no third-party libraries, no runtime bloat. The full bundle is under 5 KB gzipped.

TypeScript first

Every event payload, option, and return type is fully typed. IntelliSense works exactly the way you'd expect.

Event-driven

Subscribe to exactly the events you need. Each tracker emits strongly-typed events you can react to in real time.

Modular packages

Install only what you need — @behavier/pointer, @behavier/interaction, @behavier/page. Or grab them all with behavier.

Zero config

Works immediately with autoStart: true. Sensible defaults, fully customizable when you need more control.

Quick Start

Up and running in seconds

1

Install

$ npm install behavier
2

Import & initialize

import { createBehavier } from 'behavier'

const b = createBehavier({
  pointer: { autoStart: true },
  click:   { autoStart: true },
  scroll:  { autoStart: true },
})
3

React to events

b.pointer.on('move', ({ x, y }) => {
  console.log(`cursor: ${x}, ${y}`)
})

b.click.on('click', ({ target, count }) => {
  console.log(`${target} clicked ${count}×`)
})

b.scroll.on('scroll', ({ depthPercent }) => {
  console.log(`scrolled ${depthPercent}%`)
})
